学习yii
============================

目录结构
-------------------

      assets/             contains assets definition
      commands/           包含控制台命令类
      config/             应用配置和其他配置
      controllers/        控制器
      mail/               contains view files for e-mails
      models/             模型
      runtime/            日志 缓存
      tests/              contains various tests for the basic application
      vendor/             Yii框架自身
      views/              视图
      web/                应用根目录，包含web入口文件



安装条件
------------

PHP 5.4.0.+
